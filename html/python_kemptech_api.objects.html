create_sub_virtual_service()[source]

VirtualService factory with pre-configured LoadMaster connection

When creating a virtual service that is a sub virtual service you must pass the parent index to the constructor and mark the is_sub_vs flag as true. This will allow the save() method on the newly created subvs instance to be able to create a subvs against the parent vs. The index attribute will then be overwritten on save with the subvs’s index.
